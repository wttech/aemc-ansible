# AEM Compose - Ansible - Local Example

Test just developed Ansible modules using lifecycle playbooks.

## Prerequisites

- Mac or Linux
- Ansible 6.x and higher

# Usage 

Run command:

```shell
sh run.sh [start|provision|test|stop|destroy]
```
